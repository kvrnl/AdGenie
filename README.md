# AdGenie

**A desktop ad & social-post design studio for creators - by [KVRNL](https://kvrnl.io).**

Templates, layers, text, shapes, effects, and one-click background removal that runs entirely
on your machine - plus a built-in AI designer that builds your post live from a sentence. Make
scroll-stopping promos for OnlyFans, Fansly, X, Bluesky and more, then export to PNG, JPG, or PDF.

### Features
- Template -> drop in your photo -> style text -> export
- Layers, transform, text (fonts, bold/italic/underline, color), shapes, and effects (shadow, glow, stroke)
- **On-device background removal** - your photos never leave your computer
- Built-in **AI designer**: chat in plain language and watch it build (use your Anthropic API key or Claude Code)
- Export PNG / JPG / PDF at any size or resolution, with presets for the major platforms
